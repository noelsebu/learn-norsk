# Lær Norsk — A1 to A2

A comprehensive CEFR-aligned Norwegian language learning platform built with Next.js.

## Features

- 300+ vocabulary words across 12 categories with pronunciation & gender
- 24 essential phrases for travel and daily life
- 16 grammar topics covering the full A1–A2 CEFR curriculum
- 6 situational dialogues with toggle translations
- 3 reading comprehension passages with questions
- Interactive exercises: Quiz, Sentence Builder, Fill-in-the-Blank, Verb Conjugation
- XP progress tracking with level system
- A1/A2 level filter across all content
- Mobile-first responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Google Fonts (Crimson Pro + Nunito Sans)
